# Tailwind CSS로 시작하는 모던 웹 디자인

Tailwind CSS는 유틸리티 우선 CSS 프레임워크로, 빠른 UI 개발을 가능하게 합니다. 이 글에서는 Tailwind CSS의 기본 개념과 활용 방법을 알아보겠습니다.

## Tailwind CSS의 장점

- 빠른 개발 속도
- 일관된 디자인 시스템
- 작은 번들 사이즈
- 높은 커스터마이징 가능성

## 기본 사용법

```html
<div class="flex items-center justify-between rounded-lg bg-white p-4 shadow-md">
  <h1 class="text-2xl font-bold text-gray-800">제목</h1>
  <button class="rounded bg-blue-500 px-4 py-2 text-white hover:bg-blue-600">버튼</button>
</div>
```

## 반응형 디자인

Tailwind의 반응형 접두사를 사용하면 쉽게 반응형 디자인을 구현할 수 있습니다:

```html
<div class="w-full md:w-1/2 lg:w-1/3">
  <!-- 컨텐츠 -->
</div>
```

## 커스터마이징

tailwind.config.js 파일을 통해 테마를 커스터마이징할 수 있습니다:

```js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#1a73e8',
      },
    },
  },
};
```
