# React Hooks 패턴과 모범 사례

React Hooks는 함수형 컴포넌트에서 상태 관리와 생명주기 기능을 사용할 수 있게 해주는 강력한 기능입니다. 이 글에서는 Hooks의 다양한 패턴과 모범 사례를 살펴보겠습니다.

## Custom Hooks 설계 패턴

재사용 가능한 로직을 분리하여 Custom Hook으로 만들 수 있습니다:

```typescript
function useWindowSize() {
  const [size, setSize] = useState({
    width: window.innerWidth,
    height: window.innerHeight,
  });

  useEffect(() => {
    const handleResize = () => {
      setSize({
        width: window.innerWidth,
        height: window.innerHeight,
      });
    };

    window.addEventListener('resize', handleResize);
    return () => window.removeEventListener('resize', handleResize);
  }, []);

  return size;
}
```

## 성능 최적화 전략

불필요한 리렌더링을 방지하기 위한 전략들:

```typescript
// useMemo를 사용한 계산 결과 메모이제이션
const memoizedValue = useMemo(() => computeExpensiveValue(a, b), [a, b]);

// useCallback을 사용한 함수 메모이제이션
const memoizedCallback = useCallback(() => {
  doSomething(a, b);
}, [a, b]);
```
