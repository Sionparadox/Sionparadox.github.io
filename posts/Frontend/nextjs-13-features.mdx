# Next.js 13의 혁신적인 기능들

Next.js 13은 React 애플리케이션 개발에 혁신적인 변화를 가져왔습니다. 이 글에서는 Next.js 13의 주요 기능들을 살펴보겠습니다.

## App Router

새로운 App Router는 파일 시스템 기반의 라우팅을 제공합니다:

```tsx
// app/page.tsx
export default function Page() {
  return <h1>홈페이지</h1>;
}

// app/about/page.tsx
export default function About() {
  return <h1>소개 페이지</h1>;
}
```

## Server Components

React Server Components를 기본으로 지원하여 더 나은 성능을 제공합니다:

```tsx
// app/components/ServerComponent.tsx
export default async function ServerComponent() {
  const data = await fetchData();
  return <div>{data}</div>;
}
```

## 데이터 페칭

새로운 데이터 페칭 방식으로 더 효율적인 서버 사이드 렌더링이 가능합니다:

```tsx
async function getData() {
  const res = await fetch('https://api.example.com/data');
  return res.json();
}

export default async function Page() {
  const data = await getData();
  return <main>{/* 데이터 표시 */}</main>;
}
```

## 이미지 최적화

개선된 Image 컴포넌트로 더 나은 이미지 최적화를 제공합니다:

```tsx
import Image from 'next/image';

export default function Page() {
  return <Image src="/photo.jpg" alt="설명" width={500} height={300} priority />;
}
```
