### 문제 설명

![곰곰이](https://upload.acmicpc.net/2a25a7d6-27fd-4d7f-8e3f-410fbe3a8fd6/-/crop/199x229/70,100/-/preview/)

 크기의 커다란 격자판 위에서 곰곰이는 놀이 하나를 하고 있다. 이 격자판의 칸들 중 일부분은 빨간색으로 칠해져 있고, 나머지 칸은 노란색으로 칠해져 있다.

곰곰이는 이 격자판 위에서 뒤집기 마법을 무제한으로 사용할 수 있다. 격자판의 좌표  는 정수 를 하나 골라 뒤집기 마법을 사용하면,  의 색이 반전된다. 색이 반전된다는 것은 빨간색 칸이 노란색으로, 노란색 칸이 빨간색으로 변경되는 것을 의미한다.

![그림]()

![그림]()

현재 격자판의 상태를 뒤집기 마법만 이용하여 곰곰이가 원하는 격자판의 상태로 변경할 수 있는지 알아보자.

### 입력 

 첫 번째 줄에는 현재 격자판 상태에서의 빨간색으로 칠해진 칸의 좌표의 개수 , 곰곰이가 원하는 격자판 상태에서의 빨간색으로 칠해진 칸의 좌표의 개수  이 공백을 사이에 두고 주어진다. ()

두 번째 줄부터  개의 줄에 걸쳐, 현재 격자판 상태에서의 빨간색으로 칠해진 칸의  번째 좌표  가 공백을 사이에 두고 주어진다. ( 는 정수,  이면  이다.)

 번째 줄부터  개의 줄에 걸쳐, 곰곰이가 원하는 격자판 상태에서의 빨간색으로 칠해진 칸의  번째 좌표  가 공백을 사이에 두고 주어진다. ( 는 정수,  이면  이다.)

### 출력 

 현재 격자판의 상태를 뒤집기 마법만 이용하여 곰곰이가 원하는 격자판의 상태로 변경할 수 있다면 첫번째 줄에 YES를, 없다면 NO를 출력하라.

### 풀이
```python
import sys
input = sys.stdin.readline

N, M = map(int, input().split())
rows = [0]* 100001
cols = [0]* 100001

for _ in range(N):
    r, c = map(int, input().split())
    rows[r] += 1
    cols[c] += 1

for _ in range(M):
    r, c = map(int, input().split())
    rows[r] += 1
    cols[c] += 1

for i in range(100001):
    if rows[i] % 2 == 1 or cols[i] % 2 == 1:
        print("NO")
        exit(0)

print("YES")
    
'''
단계 1,2,3,4,5에 대해
1,2,3,4,5는 서로 변환 가능
평행이동과 뒤집기로 변환 가능해야함
평행이동 : 짝수개의 붉은칸은 평행이동 가능
뒤집기: 2가지 경우
    10   01 
    01      10
    
    11   00
    01      10
    
각 행, 열의 붉은색이 짝수인지는 변화없음.
'''
```
