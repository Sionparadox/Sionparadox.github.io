### 문제 설명

강호네 회사에는 직원이 N명이 있고, 해야할 일이 M개가 있다. 직원은 1번부터 N번까지 번호가 매겨져 있고, 일은 1번부터 M번까지 번호가 매겨져 있다.

각 직원은 자신이 할 수 있는 일들 중 한 개의 일만 담당할 수 있고, 각각의 일을 담당하는 사람은 1명이어야 한다.

각각의 직원이 할 수 있는 일의 목록이 주어졌을 때, M개의 일 중에서 최대 몇 개를 할 수 있는지 구하는 프로그램을 작성하시오.

### 입력 

 첫째 줄에 직원의 수 N과 일의 개수 M이 주어진다. (1 ≤ N, M ≤ 1,000)

둘째 줄부터 N개의 줄의 i번째 줄에는 i번 직원이 할 수 있는 일의 개수와 할 수 있는 일의 번호가 주어진다.

### 출력 

 첫째 줄에 강호네 회사에서 할 수 있는 일의 개수를 출력한다.

### 풀이
```python
import sys
sys.setrecursionlimit(10**6)
input = sys.stdin.readline

N, M = map(int, input().split())
arr = [list(map(int, input().split()))[1:] for _ in range(N)]
now = [-1]*(M+1)

def binary_matching(node):
    for work in arr[node]:
        if visited[work]: continue
        visited[work] = True
        if now[work] == -1 or binary_matching(now[work]):
            now[work] = node
            return True
    
    return False

answer = 0
for i in range(N):
    visited = [False]*(M+1)
    if binary_matching(i):
        answer += 1

print(answer)
```
