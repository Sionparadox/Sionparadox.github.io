### 문제 설명

부터 까지 개의 정점으로 이루어진 트리가 있다. 번째 간선은 서로 다른 두 정점 , 를 잇는다. ()

개의 정점 중 몇 개를 골라, 그 고른 정점들을 라고 하자. 또한, 를 만족하는  ()가 존재할 때, 정점 가 에 속한다고 부르자.

에 속하는 서로 다른 두 정점 , 에 대하여, 에 속하는 정점만을 이용하여 트리 위에서 ,  사이를 오갈 수 있다면, “와 는  위에서 연결되어 있다”고 하자.

예를 들어, 아래와 같은 트리를 생각하자. ()

![그림](https://upload.acmicpc.net/718e7c6d-b718-45a0-84b7-0ed67bd01c22/-/preview/)

만일, , 라면, “과 ”, “과 ”, “와 ”은 각각 서로  위에서 연결되어 있다. 그러나, “과 ”, “와 ”은 각각 서로  위에서 연결되어 있지 않다.

다음 조건을 모두 만족하는 정점쌍 의 개수를 의 연결 강도라고 하자.

	와 는 서로 다른 두 정점.
	.
	와 는  위에서 연결되어 있다.

고른 정점들 가 주어질 때, 의 연결 강도를 계산하는 프로그램을 작성하라. 여러분은 이러한 질의 개에 대하여 모두 답해야 한다.

### 입력 

 첫 번째 줄에 정수 이 주어진다.

다음 ()개의 줄에 각 간선에 대한 정보가 주어진다. 이 중  ()번째 줄에는 두 정수 , 가 주어진다.

다음 줄에 정수 가 주어진다.

다음 개의 줄에 각 질의에 대한 정보가 주어진다. 이 중  ()번째 줄은 번째 질의를 나타내며, 정수 와 개의 정수 가 차례대로 주어진다.

### 출력 

 첫 번째 줄부터 개의 줄에 걸쳐, 각 질의에 대한 답을 출력한다. 이 중  ()번째 줄에는 번째 질의에서 주어진 에 대하여, 의 연결 강도를 출력한다.

### 풀이
```python
import sys
from collections import deque
input = sys.stdin.readline

N = int(input())
graph = [[] for _ in range(N+1)]
for _ in range(N-1):
    u, v = map(int, input().split())
    graph[u].append(v)
    graph[v].append(u)

tree = list(range(N+1))
tree[1] = 0
queue = deque([1])
while queue:
    curr = queue.popleft()
    for nxt in graph[curr]:
        if tree[nxt] != nxt:
            continue
        tree[nxt] = curr
        queue.append(nxt)

target = [False]*(N+1)
parent = list(range(N+1))
size = [1]*(N+1)

def find(node):
    if parent[node] == node:
        return node
    parent[node] = find(parent[node])
    return parent[node]

def union(u, v):
    u, v = find(u), find(v)
    if u == v:
        pass
    if size[u] > size[v]:
        u, v = v, u
    parent[v] = u
    size[u] += size[v]
    
def rollback(arr):
    for node in arr:
        target[node] = False
        parent[node] = node
        size[node] = 1

Q = int(input())
for i in range(Q):
    _input = list(map(int, input().split()))
    K = _input[0]
    S = _input[1:]
    for node in S:
        target[node] = True
    
    for node in S:
        p = tree[node]
        if target[p]:
            union(node, p)
    
    answer = 0
    for node in S:
        if parent[node] == node:
            answer += size[node] * (size[node]-1) // 2

    print(answer)
    rollback(S)
```
