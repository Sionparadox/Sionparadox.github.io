### 입력 

 첫째 줄에 문자열 가 주어진다.

문자열 는 HG 표준 음성기호단어로 이루어질 수도 있고 아닐 수도 있다. 

모든 입력은 알파벳 소문자로만 주어진다. 

### 출력 

 주어진 문자열 가 전부 HG 표준 음성기호로 이루어 있다면, "It's HG!"를 출력하고 다음 줄에 원래 단어를 출력한다.

그렇지 않다면, 원래 단어를 알 수 없기에 "ERROR!"을 출력한다.

### 풀이
```python
import sys
input = sys.stdin.readline

mapper = {
    'a': 'aespa',
    'b': 'baekjoon',
    'c': 'cau',
    'd': 'debug',
    'e': 'edge',
    'f': 'firefox',
    'g': 'golang',
    'h': 'haegang',
    'i': 'iu',
    'j': 'java',
    'k': 'kotlin',
    'l': 'lol',
    'm': 'mips',
    'n': 'null',
    'o': 'os',
    'p': 'python',
    'q': 'query',
    'r': 'roka',
    's': 'solvedac',
    't': 'tod',
    'u': 'unix',
    'v': 'virus',
    'w': 'whale',
    'x': 'xcode',
    'y': 'yahoo',
    'z': 'zebra'
}

S = input().strip()
L = len(S)

word_list = []
idx = 0
while idx < L:
    c = S[idx]
    origin = mapper[c]
    if S.startswith(origin, idx):
        idx += len(origin)
        word_list.append(c) # 2. 리스트에 추가
    else:
        print("ERROR!")
        exit()

print("It's HG!")
print("".join(word_list))
```
