# React와 TypeScript로 시작하기

React와 TypeScript를 함께 사용하면 더 안정적이고 유지보수하기 쉬운 애플리케이션을 만들 수 있습니다. 이 글에서는 React와 TypeScript의 기본적인 사용법을 알아보겠습니다.

## TypeScript의 장점

TypeScript는 JavaScript에 정적 타입을 추가하여 다음과 같은 이점을 제공합니다:

- 타입 안정성
- 더 나은 IDE 지원
- 코드 자동완성
- 런타임 에러 감소

## 기본적인 컴포넌트 작성법

```tsx
interface Props {
  name: string;
  age: number;
}

const UserProfile: React.FC<Props> = ({ name, age }) => {
  return (
    <div>
      <h1>{name}의 프로필</h1>
      <p>나이: {age}세</p>
    </div>
  );
};
```

## 상태 관리하기

useState 훅을 사용할 때도 타입을 지정할 수 있습니다:

```tsx
const [count, setCount] = useState<number>(0);
const [user, setUser] = useState<User | null>(null);
```
